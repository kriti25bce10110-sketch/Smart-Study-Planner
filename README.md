# Smart Study Planner using Machine Learning

# Description
This project helps students manage their study schedule using Machine Learning. It predicts how much time is required to complete study tasks and suggests a daily plan.

# Features
- Predicts study completion time
- Suggests daily schedule
- Simple and beginner-friendly ML model

# Technologies Used
- Python
- Pandas
- Scikit-learn

# Installation

1. Clone the repository
2. Install dependencies:
   pip install pandas scikit-learn

# Usage

Run the program:
python main.py

Enter difficulty and study hours to get prediction.

# Output
- Predicted completion time
- Suggested study schedule
