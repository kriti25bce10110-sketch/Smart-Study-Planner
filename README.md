# Smart Study Planner using Machine Learning

# Description
This project helps students manage their study schedule using Machine Learning. It predicts how much time is required to complete study tasks and suggests a daily plan.

# Objectives
Predict study completion time using Machine Learning
Help students plan their study schedule
Demonstrate real-life application of AI & ML
Build a simple and understandable beginner project

# How It Works
- A small dataset is used to train the model
- The model learns the relationship between:
          - Subject difficulty
          - Study hours
          - Completion time
- The user enters input values
- The model predicts required study time
- AI logic gives suggestions (Easy / Moderate / Difficult)

# Features
- Predicts study completion time
- Suggests daily schedule
- Simple and beginner-friendly ML model

# Technologies Used
- Python – Programming language
- Scikit-learn – Machine Learning library
- Linear Regression – Algorithm for prediction

# Installation & Setup
1. Clone the repository:
git clone https://github.com/your-username/smart-study-planner.git
2. Navigate to project folder:
cd smart-study-planner
3. Install required libraries:
pip install -r requirements.txt

# How to Run
Run the Python file:
python main.py
Enter:
Subject difficulty (1–5)
Study hours

# Example Output
Enter subject difficulty (1-5): 4
Enter study hours: 3

Predicted Completion Time: 5.2 hours
Moderate subject, plan properly.
